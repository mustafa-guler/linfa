Expand description

A set of hyperparameters whose values have not been checked for validity. A reference to the checked hyperparameters can only be obtained after checking has completed. If the Transformer, Fit, or IncrementalFit traits have been implemented on the checked hyperparameters, they will also be implemented on the unchecked hyperparameters with the checking step done automatically.

The hyperparameter validation done in check_ref() and check() should be identical.
